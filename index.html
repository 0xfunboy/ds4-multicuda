Development Direction

⚙️

Inference Engines & LLM Optimization

Building highly optimized native inference engines for open-weight models. Focus on Metal/CUDA backends, SSD streaming, distributed inference, and directional steering for fine-grained behavior control.

🧠

Sensor-LLM Fusion

Direct projection of continuous physical state into LLM latent space. Integrating hardware sensors (IMU, temperature, voltage) with language models to create embodied AI systems that understand physical reality natively.

🔗

Multi-Provider AI Gateway

Unified access layer to consumer LLM frontends (ChatGPT, Claude, Gemini, Grok) through browser automation. Exposing them via OpenAI-compatible and Ollama-compatible APIs for seamless integration.

⛓️

Blockchain Automation

Production-grade automation for blockchain operations. Multi-wallet orchestration, retry/resume logic, dynamic fee estimation, and intelligent automation hooks for minting and transaction orchestration.

🤖

AI Agent Development

Building sophisticated AI agents with tool calling, reasoning, and long-context understanding. Focus on DeepSeek V4 Flash/Pro models with specialized tool systems and behavior control.

📊

Open-Source Research

Contributing to the open-source ML ecosystem. Building tools for model quantization, GGUF optimization, activation importance measurement, and quality testing frameworks for production-grade models.

The Vision

Building the next generation of locally-deployed AI systems that are performant, autonomous, and embodied.

I'm pursuing a path where open-weight models are optimized to run efficiently on consumer hardware, where AI systems can directly perceive and interact with physical reality through sensor fusion, and where decentralized automation on blockchains meets intelligent agent reasoning.

The goal is to create infrastructure where AI becomes a distributed, hardware-aware, and truly autonomous force for building applications that were previously impossible.
